# Bitácora de avances (por commit) — SIGCE

Esta carpeta documenta **avances por commit** para mantener trazabilidad de:

- objetivo del cambio,
- alcance y archivos impactados,
- riesgos,
- pasos de verificación.

## Índice

- `2026-03-03_e901c65_hardening_seguridad_reglas_y_uploads.md`
- `2026-03-03_695502c_refactor_container_y_server_actions.md`
- `WIP_refactor_ui_desacople_infra_app.md`

## Convención de nombres

`YYYY-MM-DD_<shortsha>_<titulo>.md`

Para cambios grandes (varios commits relacionados), se puede crear un documento `WIP_*` y luego renombrarlo al hacer el commit definitivo.
